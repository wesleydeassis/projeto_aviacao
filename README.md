# projeto_aviacao

Projeto-análise aviação

Repositório para fins de estudo, consiste na ingestão de dados com Python usando jupyter notebook, conexão com Google BigQuery e criação de Dataframes
O Projeto histórico aviação brasileira executa transformação dos dados consumidos do site https://dados.gov.br/dados/conjuntos-dados/ocorrencias-aeronauticas-da-aviacao-civil-brasileira
Onde é possível extrair diferentes arquivos CSVs das ocorrências de incidentes e acidentes na aviação brasileira. Foram utilizados dados de diferentes arquivos e realizado a  consolidação dos mesmos no Google BigQuery para serem consumidos via dashboard no power bi. O intuito foi analisar o histórico recente de acidentes fatais da aviação comercial brasileira e sua tendência.
